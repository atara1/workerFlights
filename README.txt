The command For running the project:
-clone the git repository:  https://github.com/atara1/workerFlights.git by using 
the command "git clone https://github.com/atara1/workerFlights.git"
-run in the terminal the command "npm i" for dependency installation
-run "ng serve"
-in the chrome browser insert the URL: http://localhost:4200/
