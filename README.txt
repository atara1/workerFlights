For running the project:
clone the git https://github.com/atara1/workerFlights.git
run in the terminal the command npm I for dependency installation
run ng serve
in the chrome browser insert the URL: http://localhost:4200/


The package that I install:
bootstrap
jquery for bootstrap
ag-grid-angular and ag-grid-community
ngrx/store

I use the API from the assignment:
https://interview-mock.herokuapp.com/api/workers/
https://interview-mock.herokuapp.com/api/workers/:worker_id
